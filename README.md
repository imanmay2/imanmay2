## 🎮 Contribution Shooter
 
> My GitHub contribution grid — getting blasted by a space shooter, because even commits deserve some action 🔫
 
<div align="center">
  <img src="https://raw.githubusercontent.com/imanmay2/imanmay2/output/github-contribution-grid-snake-dark.svg" alt="Contribution Shooter Animation" />
</div>
<details>
<summary>⚙️ <b>Enable this animation → Click to expand setup (2 mins)</b></summary>
<br/>
**Step 1:** In your `imanmay2` profile repo, create `.github/workflows/snake.yml`:
 
```yaml
name: Generate Snake Animation
 
on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:
 
jobs:
  generate:
    permissions:
      contents: write
    runs-on: ubuntu-latest
    timeout-minutes: 5
    steps:
      - name: Generate contribution grid snake
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - name: Push to output branch
        uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
 
**Step 2:** Go to repo **Settings → Actions → General → Workflow permissions → enable Read and write**
 
**Step 3:** Go to **Actions tab → Run workflow manually** — done! 🎉
 
</details>
